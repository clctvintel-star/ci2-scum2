The EVENT TEXT is the primary unit of analysis.

How to think about relevance:
The event is RELEVANT if a reasonable job applicant would treat it as information about "{FUND_NAME}".
Examples of relevant cases:
- explicit mention of the firm
- shorthand or alias resolved by thread context
- comparison between "{FUND_NAME}" and another firm
- follow-up comments that clearly discuss an earlier firm-specific claim
- questions about tradeoffs, pros/cons, comp, prestige, turnover, or return offers where the firm is established in the surrounding context

The event is NOT relevant only if it truly provides no usable information about "{FUND_NAME}" even after using the surrounding context.

Scoring rules:
- Use a directional score whenever the event creates a directional reputational effect.
- Do not default to 0.0 merely because the signal is mild.
- Do not default to 0.0 merely because the event contains both positives and negatives; score the net reputational effect.
- Use 0.0 when the event is genuinely neutral, balanced, ambiguous, or reputationally meaningful but directionless.

- Return null only if the event is genuinely not about "{FUND_NAME}" after considering THREAD TITLE, THREAD TEXT, and PARENT TEXT.

Important anti-abstention rule:
If the event is pertinent to "{FUND_NAME}" but reputationally weak, procedural, mixed, or mostly informational, you must still score it.
In such cases, usually return:
- sentiment = 0.0
- low confidence, but greater than 0.0
Do NOT return null for relevant-but-neutral rows.

Context inheritance rule:
For context rows, inherited meaning matters.
A short comment such as a follow-up question, reaction, or tradeoff discussion may still be about "{FUND_NAME}" if the thread title, parent text, or thread body makes that clear.
If the broader discussion is clearly firm-specific, score the local event in that inherited context.

Examples:
- Relevant but neutral -> sentiment 0.0, confidence > 0
- Relevant and weakly positive -> small positive score
- Relevant and weakly negative -> small negative score
- Not actually about the firm -> sentiment null, confidence 0.0
